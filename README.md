# React + TypeScript + Vite

# Чат-приложение (Pet-проект)

Веб-чат с функционалом, Реализована отправка текстовых и голосовых сообщений, поиск по контактам и истории, адаптивный интерфейс.

## 🚀 Демо
https://yakushev-sergey.github.io/Chat/


## 📦 Стек технологий
- **React 19** (компоненты, хуки)
- **TypeScript** (типизация)
- **Vite** (сборка)
- **CSS Modules** (стилизация)
- **gh-pages** (деплой)

## 🛠️ Функциональность
- 💬 Отправка текстовых сообщений (Enter – отправка, Shift+Enter – перенос строки)
- 🎤 Голосовые сообщения (запись, пауза, предпросмотр, отправка)
- 🔍 Поиск по контактам (фильтрация списка чатов)
- 🔎 Поиск по сообщениям с подсветкой и навигацией (▲▼)
- 😀 Вставка эмодзи через специальную кнопку
- 🤖 «Умный» бот с имитацией ответов и индикатором «печатает...»
- 📱 Адаптивная вёрстка (мобильная и десктопная версии)
- ⚡ Автоматический скролл к новым сообщениям

## 🧠 Архитектура
- Компонентный подход
- Состояние поднято в `App`
- Кастомные хуки для работы с аудио
- Типизация через `interface`
